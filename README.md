# Spinner-Data-Tracker

This is a classroom web app for running spinner experiments, logging results in real time, and aggregating them across groups to compare experimental and theoretical probability.

## How it works

1. Enter your ID (a number)


2. Choose between **Fixed** or **Custom** mode:

  - **Fixed:** 4-color spinner with preset labels (Orange, Blue, Green, Yellow). You define the number of spins.
   - **Custom:** Define your own labels and slices (2 to 15), and set the number of spins.

3. Begin recording spin results.
